Installation

Pour pouvoir utiliser le fichier sample.js, vous devez suivre les étapes ci-dessous pour vous assurer que votre environnement est correctement configuré.

Prérequis

  • Node.js version 14 ou supérieure

Dépendances

Ce projet utilise npm (Node Package Manager) pour gérer ses dépendances. Assurez-vous que npm est installé avec votre version de Node.js.

Étapes d'installation

  1. Vérifiez que Node.js est installé sur votre machine. Vous pouvez le faire en exécutant la commande suivante dans votre terminal :
    2. node -v

    Cette commande affichera la version de Node.js installée. Assurez-vous qu'elle est supérieure ou égale à v14.

  2. Installez les dépendances nécessaires en utilisant la commande suivante :
    3. npm install

    Cela téléchargera et installera toutes les dépendances requises pour le projet.

Exécution du script

Une fois l'installation terminée, vous pouvez exécuter le fichier sample.js en utilisant la commande suivante :

node sample.js

Cela affichera le message Hello world dans votre terminal.

Configuration

Pré-requis

  • Node.js v14 ou supérieur

Dépendances

  • npm (Node Package Manager) - inclus avec Node.js

Installation

Pour commencer à utiliser ce projet, vous devez d'abord installer les dépendances nécessaires. Utilisez la commande suivante dans votre terminal :

npm install

Exécution du code

Une fois les dépendances installées, vous pouvez exécuter le fichier sample.js pour voir le résultat. Le code suivant affichera "Hello world" dans la console :

console.log("Hello world");

Pour exécuter le fichier, utilisez la commande :

node sample.js

Cette commande lancera le fichier JavaScript avec Node.js et vous devriez voir le message "Hello world" s'afficher dans votre terminal.

Exploitation

Pré-requis et dépendances

  • Node.js v14+

Installation

Pour commencer à utiliser ce projet, vous devez d'abord installer les dépendances nécessaires. Utilisez la commande suivante :

npm install

Exécution du code

Une fois les dépendances installées, vous pouvez exécuter le fichier sample.js pour afficher un message dans la console. Utilisez la commande suivante :

Cette ligne de code utilise la fonction console.log pour afficher le texte "Hello world" dans la console. C'est un exemple simple qui permet de vérifier que votre environnement Node.js est correctement configuré.
